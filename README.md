# 🎫 Support Ticket Classification & Prioritization (NLP + ML)

## 📌 Project Overview
Customer support teams receive thousands of support tickets daily.  
Manually sorting these tickets causes delays, inefficiency, and missed urgent issues.

This project builds a **Machine Learning based decision-support system** that:

- Automatically classifies support tickets into categories  
- Predicts ticket priority levels  
- Helps businesses improve response time and operational efficiency  

---

## 🎯 Objectives

The system can:

✅ Read customer support ticket text  
✅ Classify ticket category (e.g., Billing, Technical Issue, Account, Refund Request)  
✅ Predict ticket priority (High / Medium / Low)  
✅ Provide automated ticket routing support  

---

## 📊 Dataset Used

**Customer Support Ticket Dataset**

Contains:

- Ticket Subject  
- Ticket Description  
- Ticket Type  
- Ticket Priority  
- Additional customer metadata

## ⚙️ Machine Learning Pipeline

### 1️⃣ Text Preprocessing
- Lowercasing  
- Stopword removal  
- Punctuation removal  
- Text cleaning using NLTK  

### 2️⃣ Feature Engineering
- TF-IDF Vectorization  

### 3️⃣ Model Training

Two classification models were trained:

- **Ticket Category Classifier**
- **Ticket Priority Predictor**

**Algorithm Used:**

- Logistic Regression
